Installazione Rapida

1. Clona il Repository

git clone https://github.com/AlexBenin01/RouteToWonderland.git
cd RouteToWonderland

2. Setup Backend

cd ChatBot_AgenziaViaggi/backend
pip install -r requirements.txt
uvicorn main:app --reload --host 0.0.0.0 --port 8000

3. Setup Frontend

cd ChatBot_AgenziaViaggi/frontend
npm install
npm run dev -- --host 0.0.0.0

4. Configurazione Database

# Importa il database
psql -U postgres -d postgres -f routeToWonderland.sql

# Installa pgvector
CREATE EXTENSION IF NOT EXISTS vector;
